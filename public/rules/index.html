ОПРЕДЕЛЕНИЕ ПОБЕДИТЕЛЕЙ АКЦИИ

Определение победителей Акции проводится среди участников, выполнивших все условия настоящих Правил.

В течение 5 (пяти) рабочих дней после ввода корректного кодового слова и прохождения регистрациивыгружается база всех участников.

База представляет собой Excel-файл, где каждая строка содержит данные участника: Ф.И.О., e-mail. Штатной функцией Excel каждой строке присваивается случайный номер в диапазоне от единицы до числа общего количества участников +1. База сортируется по колонке с присвоенными номерами, исключая первую строку с заголовками колонок. Полученные порядковые номера участвуют в определении обладателей призов, учрежденных для Акции.

Призеры определяются по формуле:

N = Q/ 4, где N – определяемое число, Q – общее количество участников Акции, 4 – общее количество призов Акции.

Если N – целое число, то призерами становятся участники с порядковыми номерами в базе, кратными N.

Если N – дробное число, то оно округляется с недостатком, и выигравшими становятся участники с порядковыми номерами в базе, кратными N.

Призы Акции распределяются среди победителей последовательно, в порядке их анонсирования в пункте «Призовой фонд» настоящих Правил. Призы, их наименование и количество анонсируются на промостранице https://cordiant.autogoda.ru/ описанием Акции.
